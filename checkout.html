Termenii de utilizare

Pentru a beneficia din plin de avantajele „Magazin Hardware”, este necesar să fiți conformat condiţiilor generale de utilizare detaliate mai jos precum şi ansamblului de legi aplicabile. Atunci când parcurgeți şi utilizați acest site, acceptați implicit şi necondiţionat termenii respectivi. Noi ne rezervăm dreptul de a actualiza prezentul set de reguli în orice moment, fără o notificare prealabilă. Modificările operate îţi sunt opozabile din momentul publicării acestora pe site. În realitate, Magazin Hardware este un proiect individual ce simulează o întreprindere (magazin online) ce se ocupă cu vinderea produselor tehnice și a pieselor de calcul.

Datele culese din formular nu se memorizează nicăieri, fiindcă acest site este rulat local fără a fi dependent pe partea serverului.

Acest site a fost realizat de la zero (from scratch), utilizănd HTML5, CSS și JavaScript cu semantică strictă. Vă aducem la cunoștință că la realizarea siteului întreg nu fost utilizată nici un framework extern (ex. Bootstrap și jQuery). Datele privind coșul de cumpărături și comportamentul siteului sunt stocate în memorie de stocare locală a browserului, care este denumită localStorage. În mod asemănător cu cookies, ele pot fi șterse din inițiativa utilizatorului.
